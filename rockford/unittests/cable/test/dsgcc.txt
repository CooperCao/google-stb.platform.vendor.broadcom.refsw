DSG-CC test utility

dsgtestclient is a simple test utility for DSG-CC functions.
It allows users to subscribe different DSG-CC client to the DSG-CC.

HOWTO build

(1) build DSG-CC application

cd BSEAV/docsis/StbHostApps/DsgCC/build
export PLATFORM=97XXX
export CROSSDEV_ROOT= real Linux toolchains directory
./build dsgcc pvnet noslim

(2) build dsgtestclient utility
./build dsgcctestclient


HOWTO run

(1) boot Linux
(2) untar Trinity release binary tarball
(3) run rc.user to bring up communication between 3255 and host
(4) start DSG-CC
		./dsgcc97xxx
(5) From another telnet session, start test app
	  ./dsgtestclient
