VC5 Driver & Tools - Release Notes

16.1

New features

  • Added support for EGL 1.5
  • Various performance improvements in a number of areas
  • Reduced the system memory consumption used by OpenGL ES 2 & 3 contexts by around 5MB each
  • GPUMonitor : object state viewing has now mostly been integrated into a single 'State' dialog
  • GPUMonitor : extra program interface data is available in the program object display
  • GPUMonitor : state for vertex array objects, buffer objects, query objects, sync objects and transform feedback objects is now displayable
  • GPUMonitor : now uses a circular buffer to record event data allowing longer run times without massive data overhead

Bug fixes

  • GL sampler objects are now shareable
  • Fix for buffer creation status using non-generated names
  • Fixed a cache flushing issue that could rarely result in a partial buffer update
  • eglquerysurface and eglsurfaceattrib weren't supporting all attributes correctly
  • Fixed an issue with scissored multisample-FBO framebuffer blits
  • Removed GL_OES_depth32 from the extension list as we don't, and never have, supported it
  • A resize after eglCreateWindowSurface(), but before any other EGL or GL calls, failed to take effect
  • Various shader compiler fixes for large shaders

15.4

New features

  • Significant performance improvements from various kernel scheduler optimizations
  • Performance increased slightly due to better handling of memory alignment
  • Added full support for Android fences
  • Added support for YUV422 EGLimage textures
  • Added support for YV12 EGLimage textures
  • Added support for EGL_EXT_client_extensions
  • Added support for EGL_EXT_platform_base extension
  • Added support for EGL_KHR_platform_android extension
  • Added support for EGL_KHR_get_all_proc_addresses extension
  • Added support for EGL_KHR_client_get_all_proc_addresses extension
  • Added support for GL_KHR_debug extension
  • Added support for GL_EXT_robustness extension
  • Added support for OES_surfaceless_context and KHR_surfaceless_context extensions
  • Added RGBA 4444 configs
  • GPUMonitor now supports BGRA textures
  • GPUMonitor now supports ASTC textures

Bug fixes

  • Fixed a problem where large buffers (>2Kx2K) could fail to render due to insufficient bin memory
  • The GLSL compiler now better schedules control flow constructs
  • Fixed a race condition during power off which could very occasionally cause a lockup
  • Fixed occasional issue with egl/glClientWaitSync for texture dependencies
  • Worked around GLSL issue where ldunif sometimes returned incorrect data following a branch uniform instruction
  • GLSL compiler now correctly allows multiple function declarations where the specification permits
  • Fixed an issue where repeated running of ogl nxclient apps caused loss of output
  • Fixed a problem running GPUMonitor with kernel mode application builds
  • GPUMonitor's performance counters are more accurately sampled for long running jobs
  • Fixed a potential race condition which could cause a stall during program termination
  • Fixed a problem with scissored framebuffer blits
  • Fixed a display issue in the Nexus platform layer which could occasionally cause tearing even in vsync mode

15.3

New features

  • Re-engineered the kernel job scheduler for up to 20% better performance in some cases
  • Added many new fast paths for texture upload and conversion

Bug fixes

  • Workaround for h/w issue causing some errant triangles in BaseMark ES3 Duel Demo
  • Workaround a number of occlusion query h/w issues
  • Workaround for corner-case h/w issue with instanced primitives
  • eglCreateWindowSurface now returns EGL_BAD_ALLOC if native window has already been used
