This release incorporates a mechanism in the build system to download Open Source Software from the Internet during the build process, rather than using a copy distributed in the Reference Software source tarball. In this release, it is used for OpenSSL, curl, and iperf, and others. In order to share downloads among local developers, and to prevent excessive downloads, downloaded items are cached in a directory named by the build system variable B_REFSW_CACHE_DIR.
