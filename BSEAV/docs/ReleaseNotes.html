This release incorporates a mechanism in the build system to download Open Source Software from the Internet during the build process, rather than using a copy distributed in the Reference Software source tarball. In this release, it is used for OpenSSL, curl, and iperf, and others. In order to share downloads among local developers, and to prevent excessive downloads, downloaded items are cached in a directory named by the build system variable B_REFSW_CACHE_DIR. The default for that directory is set to /projects/stbdev/open-source. To share downloads, either create that directory, or set B_REFSW_CACHE_DIR to another path accessible to all.

When upgrading to the included CFE v17.3 from a much earlier CFE such as CFE versions included in Phase 13.4, CFE environment variables will be lost and have to be re-programmed due to flash partition changes.

This release includes WPA Supplicant for support of the WiFi adapter internal to the 7271. The Atlas app includes WiFi, which can be activated by building with WLAN_SUPPORT=y.

atlas.encodeStart() is limited to one encode at a time.

A new tool is available to document the features of all box modes available on a platform. The code is in BSEAV/tools/bboxreport/. The bboxreport program runs on the settop, and presents output to a web browser, in a manner similar to BSEAV/tools/bmemconfig. README.txt in BSEAV/tools/bmemconfig has more information.
