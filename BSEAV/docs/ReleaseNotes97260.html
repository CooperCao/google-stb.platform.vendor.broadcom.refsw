Important notes (Top)

URSR 18.1 is the first release which contains support for 7260 B0. The B0 adds a number of new features to 7260 family including advanced HDR support and RAAGA-AX support for transcoding

Please be aware that 7260 contains a 64 bit ARM CPU. The supplied kernel comes in 32bit and 64 bit flavors. The BOLT bootloader is common to 32 and 64-bit, and can boot both of the kernels. Please refer to “Reference software 64 bit Migration Application Note” for more details.

Platforms Supported:

  • 972603/4SV - BCM72603/4 A0/B0
  • 97260SV - BCM7260 A0/B0
  • 972603/4SFF_HB - BCM72603/4 A0/B0
  • 972603/4USFF - BCM72603/4 A0/B0
  • 97260USFF - BCM7260 A0/B0
  • 972603/4DV - BCM72603/4 A0/B0

Linux kernel and root file system build instructions (Top)

  • # tar -xjf stbgcc-6.3-1.2.tar.bz2 -C /opt/toolchains
  • # export PATH=/opt/toolchains/stbgcc-6.3-1.2/bin:$PATH
  • # tar -xvjf rootfs-4.9-1.6.tar.bz2
  • # tar -xvjf stblinux-4.9-1.6.tar.bz2
  • # cd rootfs
  • # make defaults-arm64
  • # make images
  • # export LINUX=`readlink -f ../linux`

Instructions to build atlas the Nexus sample UI application (Top)

  • # tar xvfz refsw_release_97260_20180717.src.tgz
  • # export B_REFSW_ARCH=aarch64-linux
  • # export NEXUS_PLATFORM=97260
  • # export BCHP_VER=B0
  • # make -C BSEAV/app/atlas/build install

Runtime instructions (Top)

  • Connect the UHD TV via HDMI.
  • Mount the USB/SATA containing the streams as /data/videos.
  • Use the following commands to run atlas after extracting the tarball from obj.97260.B0/BSEAV/bin dir.
  • # export B_REFSW_BOXMODE=4
  • # start install
  • # start atlas
  • From the menu, select Playback -> stream name.
