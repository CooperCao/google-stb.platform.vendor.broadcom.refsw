Quick Boot instructions

  • BOLT> boot -z <network_location>:314-1.14/vmlinuz-initrd-7364a0
  • Please note the following Ethernet port assignment for different boards using BOLT 0.93. 973649SFF board has no built-in ports, therefore you must use eth1 and a USB dongle.

Set the kernel to autoboot

  • BOLT> flash -noheader <network_location>:41-1.12/vmlinuz-initrd-arm flash1.rootfs0
  • BOLT> setenv -p STARTUP "boot  flash1.rootfs0"
  • BOLT> reboot
