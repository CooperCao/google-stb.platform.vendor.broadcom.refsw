 MTSIF Cable Daughter Card Instructions

To build for the 3128 MTSIF daughter card there are modification the SV board that are needed. SPI is used to communicate to the card.

           Card Supported : . BCM3128M4PNL_V00

 

Hardware Changes.

On the daughter card, below the 3128 heat sink, RB641 and RB642 need to be changed from 4.7K ohm to zero ohm resistors.

On the 74371 SV board, do the following (for using SPI mode):

Remove R2336, R2335, RN3017.
Install R2327, R2328, R2329 and R2330.

 

To build the Software with the support enable compile flag before compiling nexus Software Stack.

#export MTSIF_CABLE_CARD=y

74371 XID board with Cable Frontend

This release contains support for HEVC 1080p30/60 decode and display. This feature can be exercised using nexus playback or atlas application.

Instructions to build atlas and use it

 

  • # tar xvfz refsw_release_974371_20170922.src.tgz
  • #cd BSEAV/app/atlas/build
  • #make install
  • Connect  TV to HDMI
  • Mount the USB/SATA containing the Movie streams as /data/videos
  • Run the following commands to run atlas
  • #start install
  • #start atlas

Boot instructions

  • BOLT> boot  server:4.1-1.12/vmlinuz-initrd-arm:

Set the kernel to autoboot

  • BOLT> flash -noheader :4.1-1.12/vmlinuz-initrd-arm flash1.rootfs0
  • BOLT> setenv -p STARTUP "boot flash1.rootfs0"
