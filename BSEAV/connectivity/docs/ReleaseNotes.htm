Notes
Notes Description
1 Default WiFi driver (if not specified): dhd
2 "Not set" means you do not have to define the NETAPP_WIFI_CHIP variable when building
3 Newer revision upgraded the WiFi chip to 43569/43570 a2 version, older versions had the a0. The default chip supported is the a2 so if you want a0 then plese set NETAPP_WIFI_CHIP=<43569a0|43570a0>
4 The board could be populted with a BCM20705 Bluetooth part, you must request this explicitly when you ESR the platform.
