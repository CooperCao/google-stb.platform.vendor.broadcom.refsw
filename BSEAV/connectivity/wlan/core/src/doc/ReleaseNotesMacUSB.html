Installing the Software

If you are installing over a previous development release, please uninstall the earlier release by hand. To uninstall, move all "USB iLine" extensions in the Extensions folder to the Trash. Unplug the USB device and reboot.

To install the USB iLine10 software, the USB device should be unplugged. Drag the "USB iLine10™ Broadcom" extension to the System Folder icon on the boot drive. A dialog will inform you that the file will be stored in the Extensions folder. Alternatively, the file can be copied directly to the Extensions folder. Plug in the USB device. There is no need to reboot after installing the file.

Internet Install

There is a great feature for USB device driver installs on Mac OS you may wish to support. When you plug in a device for which there is no driver installed, a dialog is presented to the user reading:

Drivers needed for the USB device "USB device string" are not available. Would you like to look for these drivers over the Internet?

(The "USB device string" is the string read out of the USB firmware descriptor on the device.)

This feature was introduced in Mac OS 9, so it is not available on Mac OS 8.6.

The requirements for making the Internet download available are fairly small. The process is described in the USB Software Update Technote, technote 1191, from Apple.

The summary is that someone at your company needs to register as an Apple Developer (free) at the Apple Developer web site. Apple DTS (Developer Technical Support) needs to be sent an email letting them know to associate the developer with your USB VID/PID. After that, the developer has access to the Apple USB Editor Web Page, where they can enter the record describing the device and enter the URL pointing to the driver. The record matching the device for the Software Locator is kept at Apple, but the driver itself is kept by the company owning the device. Therefore the URL points at the company's web site; <http://broadcom.com/support/drivers/usb/VIDXXXX_PIDXXXX.hqx> would be an example. (Just an example, it doesn't exist.)

To test out the driver download process before activating it on the production server, the record should be entered into the development Software Update server. Set up a test Mac to use the development sever by using the Software Locator Switch. The Software Locator Switch tool is available in the USB DDK. The USB DDK can be downloaded from the Apple SDK/DDK web page.

Using the USB iLine10 Device

When a USB iLine10 device is pugged in, a new choice, "USB iLine", will be available in the "Connect via:" pop-up menu in the AppleTalk and TCP/IP control panels. Selecting this connection will route network traffic over the USB iLine10 device.

Known Issues

  • The driver requires the Macintosh USB Manager version 1.3 or later. Version 1.3 is part of Mac OS 9 and is also built into the ROM file of certain Macs that shipped with Mac OS 8.6. The document USB Versions and Functionality on the Apple web site describes the Macintosh models and Mac OS versions that shipped with USB Manager 1.3.
  • The Mac will hang if AppleShareIP (not personal file share) is running as a file server using the USB device, AppleTalk is inactive, and the device is hot unplugged.
  • The network connection will not be usable if IPNetRouter is configured to use the USB device and is running, the device is hot unplugged, then hot replugged. The connection is usable if IPNetRouter is shutdown before the device is hot replugged.
