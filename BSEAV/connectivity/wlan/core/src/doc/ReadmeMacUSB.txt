This package contains a BixHex (.hqx) archive file for the Macintosh
and a Release Notes.html file.  Transfer the BixHex file to a
Macintosh and unpack with Stuffit Expander. The Release Notes.html
file provided is also in the Macintosh archive. Instructions for
installing the USB iLine extension are found in the release notes.
