Broadcom USB iLine10™ Network Drivers

Broadcom Corporation Confidential. For evaluation use only.

Release Contents

This binary release of the software for Broadcom's USB iLine10(tm) Network Driver consists of the Windows driver and 2 directories.

  • The top level directory contains the Windows driver and inf files.
  • qta - contains the executables for the tools to download, dump, and program the KLSI code and MAC address into the hardware.
  • obj - contains the object files for the target processor: the basic initialization and descriptors to put in the i2c; and a few small standalone diagnostics.

Tools

  • qtsarena - Displays the current memory usage. No input parameters.
  • qtsdump - Displays memory. Input parameters are address and length.
  • qtsload - Loads a program into memory and executes it. Input parameter is file name without an extension.
  • qtsi2c, qtsli2c - Program the small (qtsi2c) or large (qtsli2c) i2c eeprom. Input parameters are the file names of programs without extensions or a file name with the ".bin" extension for an image file, with plus signs between them and no spaces.
  • macparse, qtprog - Reads and updates the MAC address, vendor and device identifiers and string descriptions from the small i2c.

Target Firmware

  • i2cimg.bin - This is the image that we want to program into the i2c.
  • blink.* - A simple iLine10 chip communication test that blinks the LED1.
  • memtest.* - A Memory test program.
  • blast.* - A program that blasts packets out continuously at 2 bpb.
  • blast4.* - A program that blasts packets out continuously at 4 bpb.
  • blast8.* - A program that blasts packets out continuously at 8 bpb.
  • sink.* - A program that receives packets and discards them.
  • loopbk.* - A program that puts the iLine10 chip in loopback mode and sends and receives packets, checking them for correct contents.
  • fttest.* - A program that tests the filter table by filling it with various patterns and sending packets in loopback mode, checking whether the are received or not according with the filter rules.
