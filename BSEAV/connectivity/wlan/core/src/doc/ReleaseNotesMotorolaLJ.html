Release Number: maj.min.rcnum.incr

Broadcom Corporation Confidential.
For Evaluation Use Only.

System Requirements

Software (supports the following operating systems):

    • Linux 2.6.10

Hardware:

  • Minimum configuration: MXC300-31 Freescale Applications Processor
  • Broadcom BCM94325SDGWB SDIO Dongle Board
  • Arasan PCI SDIO Host Controller, Rev 1.2c or 1.2c2
  • SDIO Card to Mictor adapter board (BCM9SDIO2CONAD) 
  • Blue Mictor ribbon cable 

 Included in This Release

    • Dongle Firmware Image (rtecdc.bin)
    • Linux 2.6 DHD driver for SDIO (dhd.ko)
    • Linux Java API library (ljapi.so)
    • Command line utility for WD/WL command interface (wl.exe)
    • Command line utility for testing the Linux Java APIs (motolj.exe)
    • Utility to download the Dongle Firmware Image (dhd.exe)
    • Wireless Extension Tools (iwconfig.exe, iwevent.exe, iwgetid.exe, iwlist.exe, iwpriv.exe and iwspy.exe)
