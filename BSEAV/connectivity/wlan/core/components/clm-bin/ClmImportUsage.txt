USAGE

ClmImport.exe (and its 64-bit version ClmImport64.exe) is a command-line utility with the following interface:

ClmImport[64] spreadsheet_file xml_file [warnings_file] [--verbose]

spreadsheet_file - source spreadsheet file
xml_file         - resulting xml file
warnings_file    - optional output file for copy of warning  messages printed
                   to screen
--verbose        - Specify to see verbose warning messages

ClmImport.exe is 32-bit application and on certain operations recently reached memory limits. For such cases usage of ClmImport64 is recommended
