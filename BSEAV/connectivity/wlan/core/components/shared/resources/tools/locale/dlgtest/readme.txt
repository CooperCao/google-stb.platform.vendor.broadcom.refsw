
This folder contains a localization tool to test for text trunction in dialogs.

The tool (dlgtest) is built with Microsoft Visual Studio 2008:
    dlgtest_vs08.sln, dlgtest_vs08.vcproj


This tool can also be built with the following:

   1. from Microsoft Visual Studio .Net 2003, open DlgTest.sln (+dlgtest.vcproj) 
   
   2. from Microsoft Visual C++ 6.0, open dlgTest.dsw (+dlgTest.dsp)
